# TP1: Introduction Program - Welcome Class

## Description
This is a simple introductory program designed to test the development environment setup. It contains a `Welcome` class that outputs a greeting and environment details.

## Features
* Displays a welcome message to the user.
* Verifies that the compiler and runtime environment are configured correctly.
* Basic class structure following standard naming conventions.

## How to Run
1. **Clone the repository:**
   `git clone <your-repo-link>`
2. **Compile the program:**
   * For Java: `javac Welcome.java`
   * For C++: `g++ Welcome.cpp -o Welcome`
3. **Execute:**
   * For Java: `java Welcome`
   * For C++: `./Welcome`
